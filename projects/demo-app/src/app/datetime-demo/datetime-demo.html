MnDatetime Demo - Modes, Constraints & Error Handling

This demo showcases the MnDatetime component with different input modes, min/max constraints, step intervals, and error handling.

Example 1: Date Only

Uses mode: 'date' to display a date-only picker (YYYY-MM-DD).

Example 2: Time Only

Uses mode: 'time' with step: 60 for minute-level precision.

Example 3: Date & Time (Default Mode)

Default datetime-local mode combining date and time in a single input.

Example 4: Date with Min/Max Range & Custom Errors

Uses min and max to restrict the date range to 2025, with custom error messages for constraint violations.
