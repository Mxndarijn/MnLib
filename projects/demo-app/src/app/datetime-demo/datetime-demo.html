MnDatetime Demo - Modes, Variants & Validation

This demo showcases the MnDatetime component with different modes (datetime-local, date, time), size/border-radius variants, min/max constraints, and error handling.

Example 1: Datetime-local (Default Mode)

Default datetime-local mode with required validation and built-in error messages.

Example 2: Date-only Mode

Uses mode: 'date' for date-only input with larger size and border radius.

Example 3: Time-only Mode

Uses mode: 'time' with a 15-minute step interval and custom error messages.

Example 4: Min/Max Constraints & Full Width

Datetime-local with min/max constraints, full width, shadow, and small size variant.
