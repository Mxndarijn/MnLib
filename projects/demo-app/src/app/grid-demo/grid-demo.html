Basic Grid (explicit columns)

layout.cols = { base: 1, sm: 2, lg: 3 } — 1 column on phones, 2 on small screens, 3 on large.

Auto-Fit Grid (minCardWidth)

layout.minCardWidth = '16rem' — cards fill the row automatically via CSS auto-fit/minmax; no breakpoint configuration. Resize the window to compare with the grid above.

Preview Cap (maxItems) + Toolbar

layout.maxItems = 3 shows only the first three cards; toolbarTemplate hosts the count badge and a "view all" button.

Loading Skeletons

With no skeleton field the grid renders a default card placeholder. Provide a TemplateRef via skeleton to profile the exact card shape. Toggle to compare states.

Search + Client-Side Pagination
