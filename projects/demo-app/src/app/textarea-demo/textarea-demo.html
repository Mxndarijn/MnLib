MnTextarea Demo

This demo showcases the MnTextarea component with configurable rows, cols, resize behavior, and the same error handling features as MnInputField.

Example 1: Basic Textarea (vertical resize, full width)

A simple required textarea with rows: 4, resize: 'vertical', and fullWidth: true.

Example 2: Show All Errors & No Resize

showAllErrors: true, resize: 'none', with shadow and larger size. Try typing less than 10 characters to see multiple errors.

Example 3: Priority-Based Errors & Both Resize

Uses errorPriority to control which error shows first. resize: 'both' allows resizing in both directions.
